# America vs. Israel: Bombing Iran

**A fast-paced HTML5 arcade game** where America (blue jet) battles Israel (brown jet) to see who can bomb more Iranian regime buildings!  

Built in a single HTML file — no installation needed.

## 🎮 Play Now
**[Click here to play instantly →](https://tl-kls.github.io/Iranian-regime-)**

(Works perfectly on phones and desktop)

## How to Play
- **Left / Right arrows** or on-screen buttons → Move your jet  
- **SPACE** or **DROP** button → Drop bombs (you have a short cooldown)  
- Destroy as many buildings as possible before Israel does!  

**Goal**: Get the highest score margin on **Easy**, **Medium**, or **Hard** difficulty.

## Features
- Realistic jet graphics with glowing engines  
- Explosive sound effects (bomb whoosh + big booms)  
- Screen shake on hits  
- Flying debris and contrails  
- Persistent high-score per difficulty (saved in your browser)  
- Mobile-friendly controls  

## Controls
- Keyboard: Arrow keys + Space  
- Touch: On-screen buttons  

## Tech
Pure vanilla HTML5 + JavaScript + Canvas. One file, zero dependencies.
